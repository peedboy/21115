# 21115
21115 송민혁의 2025 과학 심화 탐구에 사용된 파이썬 프로그램과 아두이노 프로그램을 보관하는 레포지토리입니다.
0칸은 유동형-높이 0홀더, 1칸은 유동형-높이 1홀더...에 해당하느 데이터들이고 각 항목에 세개의 측정된 가속도
csv 파일이 있습니다. 고정형 2칸은 고정형-높이 2홀더의 측정 데이터입니다. calculate_data.py가 데이터 기반으로
운동량을 측정하는 프로그램입니다. 내부에 주석(설명)이 달려있으니 읽어 볼 수 있습니다. 탐구 보고서에 첨부한 계산
내용과 동일한 방식으로 데이터들을 계산합니다.
